# Pedra, Papel e Tesoura

Projeto desenvolvido como **atividade de curso** em JavaScript.

O objetivo da atividade foi **implementar a lógica do jogo Pedra, Papel e Tesoura** por meio da função `playRockPaperScissor`.

---

## Sobre o projeto

- A **lógica do jogo** foi desenvolvida por mim
- A **estrutura de testes automatizados e interface** foram fornecidas pelos **professores da atividade**
- Os testes servem para validar se a função retorna os resultados corretos para todas as combinações possíveis

---

## Função implementada

```js
playRockPaperScissor(player1, player2)

### Resultados esperados

- "Empate!"
- "Jogador 1 venceu!"
- "Jogador 2 venceu!"

---

## Tecnologias utilizadas

- JavaScript
- Lógica condicional
- Testes automatizados (fornecidos pela atividade)
```
